# Changelog

## v1.0.0 - 2026-07-08

Initial stable release of `create-scottish-ai-guy-app`.

- Added the initial TypeScript CLI package.
- Added interactive project creation with `@inquirer/prompts`.
- Added the template system with base and project-type templates.
- Added feature overlays for Convex, Clerk, Stripe, MDX, deterministic assistant, and blog/insights.
- Added placeholder replacement for project name, slug, project type, and year.
- Added dependency merging based on selected options.
- Added `.env.example` generation based on selected features.
- Added a practical Next.js starter template with App Router, TypeScript, Tailwind CSS, shadcn/ui-ready structure, Scottish AI Guy standards, and agent instructions.
